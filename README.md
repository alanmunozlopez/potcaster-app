This is a PodcastsHomeList App

## Getting Started

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Make build

Make build:

```bash
npm run build
# or
yarn build
# or
pnpm build
```

Run build locally:

```bash
npm run start
# or
yarn start
# or
pnpm start
```
